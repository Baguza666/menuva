# Menuva Landing Page

Landing page React + Tailwind pour **Menuva**, orientée conversion (audit / WhatsApp) pour restaurants et coffee shops.

## Lancer le projet en local

1. Installer les dépendances :

```bash
npm install
```

2. Démarrer le serveur de développement :

```bash
npm run dev
```

3. Ouvrir l’URL de preview locale :

- `http://localhost:5173/`

## Configuration rapide

- Le lien WhatsApp est centralisé dans `src/MenuvaLandingPage.jsx` :
  - `const WHATSAPP_URL = "https://wa.me/212000000000"`
- Remplacez ce numéro placeholder par le numéro réel Menuva.

## Build production

```bash
npm run build
npm run preview
```
